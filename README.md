Overview

The Book Library App is a simple React web application that helps users search for books and view book information easily.

This project was built as part of an ALX assessment to demonstrate practical frontend development skills using modern tools.

Problem

Finding books online can sometimes feel overwhelming due to complex platforms and cluttered interfaces. This app solves that by providing a clean and straightforward way to search and explore books.

Solution

The app allows users to:

Search for books by title

View book results instantly

Enjoy a fast and responsive interface

Features

Book search functionality

Dynamic rendering of book results

Responsive design for mobile and desktop

Fast performance with Vite

Tech Stack

React

Vite

JavaScript

Tailwind CSS

How to Run Locally
npm install
npm run dev
